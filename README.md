## Project Description
This project focuses on asynchronous JavaScript. I created a user directory and a form to submit new posts. Users can search by name or email, sort by name, and filter by city. The project demonstrates fetching data from APIs and updating the DOM.

## Technologies Used
- HTML  
- CSS  
- JavaScript (async/await, fetch)  
- JSONPlaceholder API  

## Features
- Display all users from JSONPlaceholder  
- Search users by name or email  
- Sort users by name (A-Z)  
- Filter users by city  
- Create a new post (title, body, user ID) and display it  

## How to Run
1. Clone this repository  
2. Open index.html in your browser
